# enum AssetType

A flag for what 'type' an Asset may store.

## Enum Values

|  |  |
|--|--|
|Font|A Font.|
|Material|A Material.|
|Mesh|A Mesh.|
|Model|A Model.|
|None|No type, this may come from some kind of invalid Asset id.|
|Shader|A Shader.|
|Solid|A Solid.|
|Sound|A Sound.|
|Sprite|A Sprite.|
|Tex|A Tex.|
